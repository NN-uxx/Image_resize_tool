# ツールの名前
Image_resize ツール

## 簡単な説明
画像ファイルの大きさを0.1倍から4倍までの間で変更します。
対応ファイルはpng,jpg,jpeg,webpの4種類です。

## 必要要件
Windows11

## 使用ライブラリー
PIL

## 使い方
1・大きさを変更したい画像ファイルを任意の場所に用意してください（面倒なら同梱のinputフォルダに入れてください。
2・ResizeTool.pyを起動します
3・UIから入力フォルダに1で用意したフォルダを選択してください
4・UIからリサイズ倍率を設定してください
5・リサイズ開始ボタンをクリックしてください

## インストール
解凍するだけ。アンインストールはフォルダごと消してください
